Welcome to the AI Agent Webcall Widget demonstration page. This widget allows you to easily integrate AI-powered webcall functionality into any website with just a few lines of code.

Features

  • Easy integration with a single script tag
  • Customizable appearance and positioning
  • Responsive design that works on all devices
  • Clean and modern UI
  • Callback support for custom logic

Quick Start

Add the following code to your HTML page:

<!-- Include the widget CSS and JS -->
<link rel="stylesheet" href="widget.css">
<script src="widget.js"></script>

<!-- Initialize the widget -->
<script>
  new AIWebcallWidget({
    position: 'bottom-right',
    primaryColor: '#007bff',
    agentName: 'AI Assistant',
    greeting: 'Hello! How can I help you today?'
  });
</script>

Configuration Options

Customize the widget to match your brand and requirements:

{
  position: 'bottom-right',        // Widget position
  primaryColor: '#007bff',         // Brand color
  buttonText: 'Talk to AI Agent',   // Button text
  agentName: 'AI Assistant',       // Agent display name
  agentAvatar: null,              // Agent avatar URL
  greeting: 'Hello!',              // Welcome message
  callEndpoint: null,             // API endpoint
  onCallStart: function() {},      // Callback
  onCallEnd: function() {},        // Callback
  onError: function(err) {}      // Error handler
}
